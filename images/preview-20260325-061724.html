スクロール連動サイドメニュー表示制御

📌 動作概要

gallery_areaが画面半分に到達してからaccess_areaが表示されるまでの間、side_areaにopenクラスを付与して表示します。

💻 コード

const galleryArea = document.querySelector(".gallery_area"); const sideArea = document.querySelector(".side_area"); const accessArea = document.querySelector(".access_area"); window.addEventListener("scroll", function () { const galleryTop = galleryArea.getBoundingClientRect().top; const accessTop = accessArea.getBoundingClientRect().top; const windowHeight = window.innerHeight; if (galleryTop < windowHeight / 2 && accessTop > windowHeight) { sideArea.classList.add("open"); } else { sideArea.classList.remove("open"); } });
