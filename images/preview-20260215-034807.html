パララックス効果の2層構造

2層構造（固定背景 + スクロール前景）

なぜ relative なのか？

✓ relative

  • スペースを占有
  • スクロール領域を生成
  • ページ高さが確保される

✗ absolute

  • フローから外れる
  • スペースを占有しない
  • スクロールバーが出ない

実装の流れ

1
背景：fixed で画面に固定
↓
2
前景：relative でスペース確保 → スクロール領域を生成
↓
3
ユーザーがスクロール → 前景だけ動く → 視差効果
HTML 
<!-- 背景（固定） -->
<div class="particle-bg"></div>

<!-- 前景（スクロール） -->
<div class="parallax-container"></div>
CSS 
.particle-bg {
  position: fixed;  /* 画面に固定 */
  z-index: -1;      /* 背���に配置 */
}

.parallax-container {
  position: relative; /* スペースを占有 */
  z-index: 1;         /* 前面に配置 */
  padding-top: 100vh; /* 最初は背景のみ表示 */
}
