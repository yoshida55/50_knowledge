① border-radius の4値は時計回り

border-radius: 左上 右上 右下 左下
border-radius: 0 20px 0 0; → 右上だけ丸くなる

② 親に overflow: hidden がないと角が白くつぶれる

❌ overflow: hidden なし
→ 角から子が はみ出る
parent { border-radius: 20px; }
✅ overflow: hidden あり
→ 角でちゃんと切れる
parent { border-radius: 20px;
  overflow: hidden; }

💡 border-radius は親の見た目を丸くするだけ。
　子要素の描画まで切ってくれるのは overflow: hidden の役割。
