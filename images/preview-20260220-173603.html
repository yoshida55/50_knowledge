📦 overflow: hidden の仕組みと使い方

💡 overflow: hidden とは？

  • 親要素からはみ出た子要素を隠すCSSプロパティ
  • 重要： 親要素に height または width の指定が必須
  • サイズ指定がないと、親は子に合わせて自動的に伸びるため効果なし
❌ 効かない height指定なし
.right_container
height: auto
子要素（300px）
親が自動で伸びる
overflow: hidden が無効
⚠️ 全て表示される
.right_container { overflow: hidden; /* height指定なし → 効かない */ }
✅ 効く height: 150px
.right_container
height: 150px
子要素（300px）
はみ出た部分が隠れる！
overflow: hidden が有効
✂️ はみ出し部分を切り取り
.right_container { overflow: hidden; height: 150px; /* ✅ 効く！ */ }
✅ 効く calc()で計算
.right_container
calc(100% - 5rem)
子要素（300px）
calc()で高さ調整
柔軟なレイアウト可能
✂️ 計算された高さで切り取り
.right_container { overflow: hidden; height: calc(100% - 5rem); /* 演算子前後にスペース必須 */ }

🎯 overflow: hidden の具体的な使用例

1. 画像のトリミング・クロップ 画像の一部だけを表示したい時。親要素で表示範囲を制限し、はみ出た部分を隠す。
2. カードコンポーネントのデザイン 角丸のカードに画像を配置する際、画像の角も丸くするために使用。
3. スライダー・カルーセル スライドする要素のうち、表示範囲外のものを隠して見せないようにする。
4. フロート解除（clearfix） float要素を含む親要素の高さを確保する古典的テクニック（現在はFlexboxが主流）。
5. テキストの1行表示 white-space: nowrap と組み合わせて、長いテキストを1行で切り取る。
6. アニメーション範囲の制限 要素が動くアニメーションで、親要��の外に出た部分を見せないようにする。
7. レイアウト崩れの防止 想定外に大きなコンテンツがレイアウトを破壊するのを防ぐ。
