the_post_navigation()

WordPressの単一投稿ページで、前の記事と次の記事へのナビゲーションリンクを表示する関数

基本的な使い方

<?php the_post_navigation(); ?>

生成されるHTML構造

実際の表示イメージ

← 前の投稿
次の投稿 →

パラメータ付き使用例

<?php the_post_navigation( array( 'prev_text' => '← %title', 'next_text' => '%title →', 'screen_reader_text' => '投稿ナビゲーション' ) ); ?>

主な特徴

  • 単一投稿専用: single.phpで使用
  • 自動生成: HTMLマークアップを自動で出力
  • 前後の記事: 公開日順に前後の投稿へリンク
  • CSSクラス: .post-navigation, .nav-previous, .nav-next
