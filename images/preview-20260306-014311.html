🖼️ 階段式ギャラリーレイアウト

text-align と nth-child を使った左右交互配置の仕組み

48%
6rem
階段式ギャラリー - リアルタイムプレビュー
  • 奇数（1番目）
  • 偶数（2番目）
  • 奇数（3番目）
.gallery_item {
margin-bottom: 6rem;
}

.gallery_item:nth-child(odd) {
text-align: left;
}

.gallery_item:nth-child(even) {
text-align: right;
}

.gallery_img {
width: 48%;
}
💡 仕組みの解説：
• :nth-child(odd)は奇数番目の要素（1,3,5...）を選択し、text-align: leftで画像を左寄せ
• :nth-child(even)は偶数番目の要素（2,4,6...）を選択し、text-align: rightで画像を右寄せ
• 画像のwidthを48%にすることで、左右に余白が生まれ階段状に見える
• margin-bottomで各画像間の縦方向の間隔を調整
