background-attachment: fixed とは？

背景画像をビューポート（画面）に固定し、スクロールしても背景が動かない効果を実現します。視差効果（パララックス）の実装に使用されます。

方法1: CSSで実装

CSS 
.middle-section {
  background-image: url('image.jpg');
  background-attachment: fixed;
  background-position: center;
  height: 500px;
}
注意： 背景はビューポート全体に対して固定されるため、セクション位置に合わせて background-position の調整が必要です。

方法2: JavaScriptで途中から固定

JavaScript 
window.addEventListener('scroll', () => {
  const scrolled = window.pageYOffset;
  const section = document.querySelector('.middle-section');
  
  // セクションが画面内に入ったら背景位置を調整
  section.style.backgroundPositionY = (scrolled * 0.5) + 'px';
});
実務では： セクションごとに異なる背景を固定したい場合、CSSの fixed よりもJavaScriptや transform を使った実装の方が制御しやすいことが多いです。
