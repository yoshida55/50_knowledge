🎯 Flexbox min-width: 0 問題デモ

❌ 問題: min-width: auto (デフォルト)
長いコンテンツテキスト
flex: 1
親要素の幅: 600px
.b の実際の幅: -
.b の min-width: auto (コンテンツ幅)
.c の実際の幅: -
⚠️ 問題: .b はコンテンツより小さくならないため、親要素を縮めても flex: 2 の比率が守られません。
✅ 解決: min-width: 0
長いコンテンツテキスト
flex: 1
親要素の幅: 600px
.b の実際の幅: -
.b の min-width: 0 (強制的に縮小可能)
.c の実際の幅: -
✓ 解決: min-width: 0 により、.b はコンテンツより小さくなれるため、flex: 2 と 1 の比率 (2:1) が正しく維持されます。
