📌 視差効果の仕組み

背景をposition: fixedで固定し、前景を通常スクロールさせることで、奥行きのある立体的な表現を実現しています。

🎨 背景レイヤー

画面に固定されたまま動かず、スクロールしても常に同じ位置に表示されます。

.particle-bg { position: fixed; z-index: -1; }

📄 前景レイヤー

通常通りスクロールし、z-indexで背景��前面に配置されます。

.parallax-container { position: relative; z-index: 1; padding-top: 100vh; }
