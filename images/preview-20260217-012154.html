🎨 CSS Transition × Transform デモ

📱 実際の動作デモ

🏠 ヘッダー要素 position: fixed

ページコンテンツ

下のボタンをクリックする���、ヘッダーが左にスライドして隠れます。

この動きは transform: translateX(-100%) と transition: 0.3s で実現されています。

⬅️

💡 仕組み

通常時: transform: translateX(0) でヘッダーは表示されています

hidden時: transform: translateX(-100%) でヘッダーが左に自分の幅分移動します

transition: この変化を0.3秒かけて滑らかにアニメーションします

💻 実装コード

/* 基本スタイル */
.header {
position: fixed;
background-color: white;
transition: transform 0.3s ease-in-out; /* 滑らかな動き */
}
/* 非表示状態 */
.header.hidden {
transform: translateX(-100%); /* 左に100%移動 */
}
// JavaScript
const header = document.querySelector('.header');
header.classList.toggle('hidden'); // hiddenクラスを切り替え
