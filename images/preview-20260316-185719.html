WordPress 固定ページの仕組み

📊 データの流れ

📌 重要ポイント

  • DBに保存される → PHPファイルは自動で作られない
  • テンプレートは自分で作る → page-{スラッグ}.php をテーマに配置
  • 命名規則が重要 → スラッグが about なら page-about.php
  • 自動マッチング → URL /about/ と page-about.php が自動で紐づく

💻 実装例

<?php
/**
 * Template Name: About Page
 * page-about.php
 */
get_header(); ?>

<main class="page-content">
    <?php if ( have_posts() ) : while ( have_posts() ) : the_post(); ?>
        
        <h1><?php the_title(); ?></h1>
        
        <div class="entry-content">
            <?php the_content(); ?>
        </div>
        
    <?php endwhile; endif; ?>
</main>

<?php get_footer(); ?>
