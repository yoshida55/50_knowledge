4. 重要ポイント解説

★ inset()の方向は直感と逆

inset(top right bottom left)

inset(100% 0 0 0) = 「上から100%削る」→ 下から見え始める

理由：topが100%の場合、要素の上端から100%の位置まで削除される = 全体が削除される。そこから0%に戻すと、上から順に復元される = 下から上に表示されるように見える。

★ transition vs @keyframes

transition： A → B の1方向のみ。シンプルな変化に最適。

@keyframes： 複数ステップ可能。「消えて→再表示」のような複雑な動きに必須。

今回のアイコンアニメーション：0% 表示 → 1% 消去 → 100% 再表示 の3段階が必要なため、@keyframesを使用。
