background-attachment: fixed

背景画像をビューポートに固定し、スクロールしても背景が動かないプロパティです。視差効果（パララックス）の実装に使用されます。

動作の仕組み

視覚デモ

コンテンツエリア
背景は固定、このエリアだけスクロール
⬆️ 背景固定・コンテンツ移動のイメージ

実装コード

.hero {
  background-image: url('image.jpg');
  background-attachment: fixed;
  background-size: cover;
  background-position: center;
}

結果：ページをスクロールしても背景画像はビューポートに固定され、コンテンツだけが上下に移動します。
