例：左→右にリビールなら右を100%、上→下にリビールなら下を100%にしてから0%にする

💡 inset()の値の順序と仕組み

inset(top right bottom left) の順序で、各辺からどれだけ内側に切り取るかを指定します。

リビール方向の法則：

✅ 左→右にリビール：inset(0 100% 0 0) → inset(0 0 0 0)（右辺を100%→0%）

✅ 右→左にリビール：inset(0 0 0 100%) → inset(0 0 0 0)（左辺を100%→0%）

✅ 上→下にリビール：inset(0 0 100% 0) → inset(0 0 0 0)（下辺を100%→0%）

✅ 下→上にリビー���：inset(100% 0 0 0) → inset(0 0 0 0)（上辺を100%→0%）
