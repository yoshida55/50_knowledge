Flexbox 2カラムレイアウト解説

親要素にdisplay: flexを指定するだけで、左右2カラムのレイアウトを実現できます。

実際の動作デモ

記事一覧（左カラム）

メインコンテンツがここに表示されます

サイドバー（右カラム）

執筆者プロフィール
ランキング
アーカイブ
ポイント: 親要素の.contents_areaにのみflexを適用。子要素は自動的に横並びになります。

CSSコード

.contents_area {
  display: flex;
  justify-content: space-between;
}

.contents_main  { width: 65%; }
.contents_sidebar { width: 33%; }
