## MDファイルをまとめる時

今後、MDファイルをまとめる際は、以下のルールを厳守してください。

1. タイトル: 全体のタイトルのみ ## を使用する。
2. サブタイトル: 項目名を ` で囲む。
3. コードブロック: 必ず言語名（html, css, javascript）を明示する。

◯JavaScriptの場合以下で囲む
```javascript
```
◯htmlの場合
```html
```
◯cssの場合
```css
```
