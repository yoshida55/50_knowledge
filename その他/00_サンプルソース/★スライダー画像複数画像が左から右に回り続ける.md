# slick スライダー パターン集

slick carousel を使ったスライダーのパターンまとめ。
jQuery + slick が必要（CDNは末尾の補足を参照）。

---

## 🗂 パターン一覧（どれを使うか迷ったらここで選ぶ）

```
① img直接・centerMode（最もシンプル）
   ┌──┐ ┌────────┐ ┌──┐
   │1 │ │   2    │ │3 │
   └──┘ └────────┘ └──┘
          ↑中央が広い

② slider_item div あり（カードにCSSを当てたいとき）
   ┌────┐ ┌────┐ ┌────┐
   │ 1  │ │ 2  │ │ 3  │
   └────┘ └────┘ └────┘

③ 右端に半分チラ見せ（Nextで1枚ずつ動く）
   ┌────┐ ┌────┐ ┌────┐ ┌──
   │ 1  │ │ 2  │ │ 3  │ │4…
   └────┘ └────┘ └────┘ └──

④ 隙間なし（画像がぴったりくっつく）
   ┌────┐┌────┐┌────┐
   │ 1  ││ 2  ││ 3  │
   └────┘└────┘└────┘
     ↑ margin: 0 で隙間ゼロ
```

---

### ① img直接・centerMode（最もシンプル） ★★★

`.slider` の直接の子要素が1枚のスライドになる。`slider_item` div は不要。
画像だけ並べればすぐ動く。

**HTML**
```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8" />
  <link rel="stylesheet" href="css/style.css" />
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/slick-carousel@1.8.1/slick/slick.css" />
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/slick-carousel@1.8.1/slick/slick-theme.css" />
</head>
<body>

  <div class="slider">
    <img src="img/slide01.jpg" alt="" />
    <img src="img/slide02.jpg" alt="" />
    <img src="img/slide03.jpg" alt="" />
    <img src="img/slide04.jpg" alt="" />
  </div>

  <button type="button" class="my_btn">Next →</button>

  <!-- 読込順：jQuery → slick → 自分のJS（この順番厳守） -->
  <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
  <script src="https://cdn.jsdelivr.net/npm/slick-carousel@1.8.1/slick/slick.min.js"></script>
  <script src="js/work.js"></script>
</body>
</html>
```

**CSS（css/style.css）**
```css
.slider img {
  width: 100%;
  display: block;
}

/* slick が自動生成するクラス（HTMLに書く必要なし） */
.slider .slick-slide {
  margin: 0 0.5rem; /* 画像間の隙間 */
}
```

**JavaScript（js/work.js）**
```javascript
$(function () {
  $(".slider").slick({
    arrows: false,
    autoplay: false,
    slidesToShow: 3,     // 常に3枚表示。幅は自動で均等割り
    centerMode: true,
    centerPadding: "3rem", // 両端のチラ見え幅（大きくすると両端が小さくなる）
    responsive: [
      {
        breakpoint: 768,
        settings: {
          slidesToShow: 1,
          centerPadding: "5rem",
        },
      },
    ],
  });

  $(".my_btn").on("click", function () {
    $(".slider").slick("slickNext");
  });
});
```

### ② slider_item div あり（カードにCSSを当てたいとき）

画像をdivで囲むことで、カードごとに角丸・影などのスタイルを当てやすくなる。
パターン①より少し記述が増えるが、デザインの自由度が高い。

**HTML**
```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8" />
  <link rel="stylesheet" href="css/style.css" />
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/slick-carousel@1.8.1/slick/slick.css" />
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/slick-carousel@1.8.1/slick/slick-theme.css" />
</head>
<body>

  <div class="slider">
    <div class="slider_item"><img src="img/slide01.jpg" alt="" /></div>
    <div class="slider_item"><img src="img/slide02.jpg" alt="" /></div>
    <div class="slider_item"><img src="img/slide03.jpg" alt="" /></div>
    <div class="slider_item"><img src="img/slide04.jpg" alt="" /></div>
  </div>

  <!-- type="button" 必須。省略するとページリロードされる -->
  <button type="button" class="my_btn">View More →</button>

  <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
  <script src="https://cdn.jsdelivr.net/npm/slick-carousel@1.8.1/slick/slick.min.js"></script>
  <script src="js/work.js"></script>
</body>
</html>
```

**CSS（css/style.css）**
```css
/* slick が自動生成するクラスに margin を指定（gap の代替） */
.slider .slick-slide {
  margin: 0 0.5rem;
}

/* 各カード */
.slider_item img {
  width: 100%;
  height: auto;
  display: block;
}
```

**JavaScript（js/work.js）**
```javascript
$(function () {
  // ① slick 初期化（必ず先に書く）
  $(".slider").slick({
    arrows: false,
    autoplay: false,
    slidesToShow: 3,
    centerMode: true,
    centerPadding: "3rem",
    responsive: [
      {
        breakpoint: 768,
        settings: {
          slidesToShow: 1,
          centerPadding: "5rem",
        },
      },
    ],
  });

  // ② クリックイベント（初期化の後に書く）
  $(".my_btn").on("click", function () {
    $(".slider").slick("slickNext"); // 次へ
    // $(".slider").slick("slickPrev"); // 前へ
  });
});
```

**a タグをボタンに使う場合は `e.preventDefault()` が必要**
```javascript
$(".my_btn").on("click", function (e) {
  e.preventDefault(); // href="#" でページトップに飛ぶのを防ぐ
  $(".slider").slick("slickNext");
});
```

### ③ 右端に半分チラ見せ（Nextで1枚ずつ動く）

```
┌────┐ ┌────┐ ┌────┐ ┌──
│ 1  │ │ 2  │ │ 3  │ │4（半分）
└────┘ └────┘ └────┘ └──
         ↓ Next押す
     ┌────┐ ┌────┐ ┌────┐ ┌──
     │ 2  │ │ 3  │ │ 4  │ │5（半分）
     └────┘ └────┘ └────┘ └──
```

- `slidesToShow: 3.5` → 小数点で半分チラ見せできる
- `centerMode: false` → 左端から並べる（centerModeはオフ必須）
- 親要素に `overflow: hidden` → はみ出た分を隠す

**HTML**
```html
<section class="member_area">
  <div class="slider">
    <img src="img/slide01.jpg" alt="" />
    <img src="img/slide02.jpg" alt="" />
    <img src="img/slide03.jpg" alt="" />
    <img src="img/slide04.jpg" alt="" />
    <img src="img/slide05.jpg" alt="" />
  </div>
  <button type="button" class="my_btn">Next →</button>
</section>
```

**CSS**
```css
.member_area {
  overflow: hidden; /* はみ出た4枚目を隠す（必須） */
}
```

**JavaScript**
```javascript
$(function () {
  $(".slider").slick({
    arrows: false,
    autoplay: false,
    slidesToShow: 3.5,   // 3枚 + 半分
    slidesToScroll: 1,   // 1枚ずつ動く
    centerMode: false,   // オフ必須
    infinite: false,     // 端まで行ったら止まる
  });

  $(".my_btn").on("click", function () {
    $(".slider").slick("slickNext");
  });
});
```

## 補足

### Q. CDN（外部ライブラリ）は何が必要？
→ slick は jQuery ベースのため、3つ必要。HTMLの `<head>` と `</body>` 直前に書く。

```html
<!-- head に書く（CSS） -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/slick-carousel@1.8.1/slick/slick.css" />
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/slick-carousel@1.8.1/slick/slick-theme.css" />

<!-- </body> 直前に書く（JS）。順番厳守：jQuery → slick → 自分のJS -->
<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/slick-carousel@1.8.1/slick/slick.min.js"></script>
<script src="js/work.js"></script>
```

### Q. `.slick-slide` ってHTMLに書くの？
→ 書かなくていい。slick が自動で生成するクラス。
検証ツールで確認すると、実際に追加されているのが見える。

### Q. `centerPadding` と CSS の `padding` は何が違う？
→ 役割がまったく違う。

| | 効果 |
|--|--|
| `centerPadding` | 両端の画像をチラ見せする幅。大きくすると中央の画像が小さくなり、横がよく見える |
| CSS `padding` | 画像の内側の余白。均等な隙間を作るときに使う |

両方を組み合わせることが多い。値は同じ数値にすることが多い。
