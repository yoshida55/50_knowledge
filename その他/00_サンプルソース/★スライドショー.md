

```html
<div class="bg_image">
  <div class="slide slide1"></div>
  <div class="slide slide2"></div>
  <div class="slide slide3"></div>
</div>
```

```css
/* 外枠（コンテナ）の設定 */
.bg_image {
  width: calc(100% - 36rem);
  height: 72rem;
  position: relative;
  overflow: visible;

  border: 1px solid #fff;
  box-sizing: border-box; /* 枠線をサイズに含める（レイアウト崩れ防止） */
}

/* 内側のぼかし（オーバーレイ）の設定 */　　★画像の上にぼかしをかさねたいがため、afterを使用している。
.bg_image::after {
  content: "";
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 72rem;

  /* ▼修正2：ここは「影（ぼかし）」だけにする */
  /* 数字を調整して「淡さ」を決める（0px 0px 50px 20px くらいが程よい） */
  box-shadow: inset 0px 0px 20px 20px #fff;　★これをかぶせるため

  z-index: 10;　　
  pointer-events: none;
}

.slide {
  width: 100%;
  height: 72rem;
  position: absolute;　★これがないと、slide1～3まで合計２１６の高さに、なるこれがあることによって、同じサイズ72remになる。
  opacity: 0;
  transition: opacity 1s ease-in-out;
}

.slide1 {
  animation: slideshow 15s infinite;
  background-image: url("../img/mainvisual1.jpg");
  background-size: cover; /* 要素いっぱいに広げる */
  background-position: center; /* 中央基準で表示 */
  animation-delay: 0s;
}

.slide2 {
  animation: slideshow 15s infinite;
  background-image: url("../img/mainvisual2.jpg");
  background-size: cover; /* 要素いっぱいに広げる */
  background-position: center; /* 中央基準で表示 */
  animation-delay: 5s;
}

.slide3 {
  animation: slideshow 15s infinite;
  background-image: url("../img/mainvisual3.jpg");
  background-size: cover; /* 要素いっぱいに広げる */
  background-position: center; /* 中央基準で表示 */
  animation-delay: 10s;
}

★
・3枚の画像が同じ場所に重なっている
・animation-delay で表示タイミングをずらす
・opacity で1枚ずつ表示/非表示を切り替える

★合計時間（15秒）を100%として、1枚の画像が「いつ出て、いつ消えるか」を指示しています。
@keyframes slideshow {
  0% {
    opacity: 0;
  }
  5% {
    opacity: 1;
  }
  33% {
    opacity: 1;
  }
  38% {
    opacity: 0;
  }
  100% {
    opacity: 0;
  }
}
```
