# パララックス

```
┌──────────────────────────┐
│  🏠 固定背景（動かない）  │ ← position: fixed
│                          │
│  ┌──────┐ ┌──────┐       │ ← 前景（スクロールで動く）
│  │photo1│ │photo2│       │
│  └──────┘ └──────┘       │
└──────────────────────────┘
  スクロールするごとに前景だけ動く → 奥行き感が出る
```

**ポイント**
- 背景は `position: fixed` で固定
- 前景に `data-speed="0.3"` などを指定 → JSで速さを変える
- ライブラリなし・JavaScriptのscrollイベントだけで実現

**構造の考え方**
```
① .particle-bg（固定背景）   ← fixed / z-index: -1
② .parallax-container       ← relative / z-index: 1
   └── .parallax-section
         └── .photo-box（data-speed="0.3"）← JSで動く
```

```html
<head>
  <meta charset="UTF-8" />
  <link rel="stylesheet" href="css/パララックス.css" />
</head>
<body>
  <!-- ①固定背景（絵の家） -->
  <div class="particle-bg">
    <div class="house-illustration"></div>
  </div>

  <!-- ②スクロールするコンテンツ -->
  <div class="parallax-container">
    <!-- セクション1 -->
    <section class="parallax-section">
      <div class="photo-box photo1" data-speed="0.3"></div>
      <div class="photo-box photo2" data-speed="0.5"></div>
    </section>
    <!-- セクション2 -->
    <section class="parallax-section">
      <div class="photo-box photo3" data-speed="0.4"></div>
      <div class="photo-box photo4" data-speed="0.6"></div>
    </section>
  </div>

  <script src="js/パララックス.js"></script>
</body>
</html>
```

**CSS**
```css
body {
  margin: 0;
  overflow-x: hidden;
}

/* ①固定背景 */
.particle-bg {
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  z-index: -1;
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  display: flex;
  justify-content: center;
  align-items: center;
}

/* お家の胴体 */
.house-illustration {
  width: 300px;
  height: 300px;
  background: rgba(255, 255, 255, 0.1);
  border: 3px solid rgba(255, 255, 255, 0.3);
  position: relative;
}

/* お家の屋根（::before で三角形） */
.house-illustration::before {
  content: "";
  position: absolute;
  top: -80px;
  left: 50%;

  width: 0;
  height: 0;
  border-left: 180px solid transparent;
  border-right: 180px solid transparent;

```

**JavaScript**
```javascript
window.addEventListener("scroll", () => {
  const scrollY = window.scrollY;

  document.querySelectorAll(".photo-box").forEach((box) => {
    const speed = parseFloat(box.dataset.speed) || 0.5;
    // スクロールと逆方向に動かす（マイナス）
    box.style.transform = `translateY(${-scrollY * speed}px)`;
  });
});
```
