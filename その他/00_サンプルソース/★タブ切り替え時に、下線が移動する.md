# タブ切り替え時に、下線が移動する

クリックしたタブに下線がスライドして移動するUI。offsetLeft+offsetWidthで位置を取得し、CSSのtransitionでなめらかに動かす。

★核心：JSは位置を変えるだけ。「なめらかに動く」のはCSSのtransitionの仕事。
```js
line.style.left = activeBtn.offsetLeft + 'px';
line.style.width = activeBtn.offsetWidth + 'px';
```

## ★重要ポイント

【★ポイント1: scriptは</body>直前に書く】
```html
<script src="js/tab.js"></script>
</body>
```
HTMLが全部読み込まれてからJSを動かすため。先に書くと要素が見つからずエラーになる。

【★ポイント2: 線の位置はoffsetLeft/offsetWidthで取得】
```js
function moveLine(activeBtn) {
  line.style.left = activeBtn.offsetLeft + 'px';  // 左端の位置
  line.style.width = activeBtn.offsetWidth + 'px'; // 幅
}
```
`offsetLeft` = 親要素からの左端の距離（px）
`offsetWidth` = 要素の横幅（px）

【★ポイント3: スライドアニメーションはCSSのtransition】
```css
transition: left 0.3s ease, width 0.3s ease;
```
数字を変えると速さが変わる。`0.1s`=速い / `1s`=ゆっくり

【★ポイント4: タブ切り替えはclassListでis-active/is-showを付け替えるだけ】
```js
buttons.forEach(b => b.classList.remove('is-active'));
btn.classList.add('is-active');

contents.forEach(c => c.classList.remove('is-show'));
document.querySelector(`[data-content="${target}"]`).classList.add('is-show');
```

## HTML
```html
<div class="tab_wrapper">
  <div class="tab_list">
    <button class="tab_btn is-active" data-tab="news">ニュース</button>
    <button class="tab_btn" data-tab="press">プレスリリース</button>
    <span class="tab_line"></span>
  </div>

  <div class="tab_content is-show" data-content="news">
    <p>ニュースの内容</p>
  </div>
  <div class="tab_content" data-content="press">
    <p>プレスリリースの内容</p>
  </div>
</div>

<script src="js/tab.js"></script>
```

## CSS
```css
.tab_list {
  position: relative; /* 線の基準点 */
  display: flex;
  border-bottom: 1px solid #ccc;
}

.tab_btn {
  padding: 12px 24px;
  background: none;
  border: none;
  cursor: pointer;
  font-size: 14px;
}

/* 動く線 */
.tab_line {
  position: absolute;
  bottom: 0;
  height: 2px;
  background: #000;
  transition: left 0.3s ease, width 0.3s ease;
}

/* 非表示コンテンツ */
.tab_content {
  display: none;
}
.tab_content.is-show {
  display: block;
}
```

## JavaScript
```javascript
const buttons = document.querySelectorAll('.tab_btn');
const line = document.querySelector('.tab_line');
const contents = document.querySelectorAll('.tab_content');

// 最初の線の位置をセット
moveLine(document.querySelector('.tab_btn.is-active'));

buttons.forEach(btn => {
  btn.addEventListener('click', () => {
    // アクティブクラスを切り替え
    buttons.forEach(b => b.classList.remove('is-active'));
    btn.classList.add('is-active');

    // 線を動かす
    moveLine(btn);

    // コンテンツを切り替え
    const target = btn.dataset.tab;
    contents.forEach(c => c.classList.remove('is-show'));
    document.querySelector(`[data-content="${target}"]`).classList.add('is-show');
  });
});

function moveLine(activeBtn) {
  line.style.left = activeBtn.offsetLeft + 'px';
  line.style.width = activeBtn.offsetWidth + 'px';
}
```

---

## 動作フロー
1. ページ読み込み時 → 最初のタブに線をセット
2. タブクリック → `is-active` を付け替え
3. `moveLine()` で線の `left` と `width` を更新
4. CSS の `transition` でなめらかにスライド
5. `data-content` と一致するコンテンツを `is-show` で表示切り替え
