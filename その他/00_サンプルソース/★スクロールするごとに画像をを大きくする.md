# スクロールで画像拡大 - flex center + flex-shrink 0 + overflow-x hidden のセットで中央から均等に広がる

スクロール量に応じて3枚の画像が中央から均等に拡大する。PC/SP対応。

★3つのCSSプロパティはセットで使う（これがないと右寄りに広がる）
```css
justify-content: center;   /* 中央揃え */
flex-shrink: 0;            /* 100%超えても縮まない */
overflow-x: hidden;        /* はみ出した左右を切り取る */
```

## ★重要ポイント

【★ポイント1: 3つのCSSはセットで使う】
```css
.parent {
  display: flex;
  justify-content: center;   /* center にする（space-between だと隙間ができる） */
  overflow-x: hidden;        /* はみ出した左右を非表示にする */
}
.child-img {
  flex-shrink: 0;            /* 合計100%超えても縮まない → overflow-x が切り取る */
}
```
この3つが揃って初めて「中央から均等に広がる」動きになる

【★ポイント2: セレクタは親ではなく画像（子）に指定する】
```javascript
// ❌ 親に指定 → セクション全体が33%に縮む
$(".main_area").css("width", ...);

// ✅ 画像（子）に指定 → 画像だけが変わる
$(".main_logo_img").css("width", ...);
```
親のwidthを変えると中の要素全部が縮む。子のwidthを変えればその要素だけが変わる

【★ポイント3: `100/3` で割り算する（`33`と書かない）】
```javascript
// ❌ 33 × 3 = 99% → 1%の隙間ができる
const newWidth = 33 + scrollTop / 10;

// ✅ 100/3 × 3 = 100% → ぴったり
const newWidth = 100 / 3 + scrollTop / 10;
```

## HTML
```html
<section id="mainvisual">
  <div class="mainvisual_imgwrapper">
    <img class="mainvisual_img" src="img/mainvisual1.jpg" alt="花1" />
    <img class="mainvisual_img" src="img/mainvisual2.jpg" alt="花2" />
    <img class="mainvisual_img" src="img/mainvisual3.jpg" alt="花3" />
  </div>
</section>
```

## CSS
```css
/*
  構造:
  <section #mainvisual>                    ← セクション全体
    <div .mainvisual_imgwrapper>           ← flex親（中央揃え + はみ出し非表示）
      <img .mainvisual_img>               ← 画像（3枚並び・縮まない）
      <img .mainvisual_img>
      <img .mainvisual_img>
*/

#mainvisual {
  width: 100%;
  height: 100vh;
}

.mainvisual_imgwrapper {
  width: 100%;
  height: 100%;
  display: flex;
  justify-content: center;    /* ★ 中央揃え（space-between ではない） */
  overflow-x: hidden;         /* ★ はみ出した左右を切り取る */
}

.mainvisual_img {
  width: calc(100% / 3);      /* ★ 33%ではなく 100/3 で正確に */
  height: 100vh;
  object-fit: cover;
  flex-shrink: 0;             /* ★ 合計100%超えても縮まない */
}
```

## JavaScript（vanilla JS版）
```javascript
// スクロールで3枚の画像を拡大（PC）/ 縮小（SP）
window.addEventListener("scroll", function () {
  const scrollTop = window.scrollY;
  const imgs = document.querySelectorAll(".mainvisual_img");

  if (window.innerWidth > 900) {
    // PC: スクロール量 ÷ 10 を幅に加算
    imgs.forEach(function (img) {
      img.style.width = 100 / 3 + scrollTop / 10 + "%";
    });
  } else {
    // SP: スクロール量 ÷ 10 を幅から減算
    imgs.forEach(function (img) {
      img.style.width = 100 - scrollTop / 10 + "%";
    });
  }
});
```

## JavaScript（jQuery版）
```javascript
// スクロールで3枚の画像を拡大（PC）/ 縮小（SP）
$(window).on("scroll", function () {
  const scrollTop = $(this).scrollTop();

  if (window.innerWidth > 900) {
    // PC: スクロール量 ÷ 10 を幅に加算
    $(".mainvisual_img").css("width", 100 / 3 + scrollTop / 10 + "%");
  } else {
    // SP: スクロール量 ÷ 10 を幅から減算
    $(".mainvisual_img").css("width", 100 - scrollTop / 10 + "%");
  }
});
```

## 動作フロー
1. ページ読み込み → 3枚が `calc(100%/3)` で均等に並ぶ
2. スクロール → 各画像の `width` が増える（PC）/ 減る（SP）
3. 合計が100%を超える → `flex-shrink: 0` で縮まない
4. `overflow-x: hidden` がはみ出した左右を切り取る
5. `justify-content: center` のおかげで中央から均等に広がって見える

## トラブルシューティング
### 症状: 画像が右寄りに広がる
**原因**: `justify-content: space-between` になっている / `overflow-x: hidden` がない
**対処**: `center` + `overflow-x: hidden` + `flex-shrink: 0` の3点セットを確認

### 症状: 画像の間に黒い隙間ができる
**原因**: widthを `33` と書いている（33% × 3 = 99%）
**対処**: `100 / 3` と書く（33.333...% × 3 = 100%）

### 症状: セクション全体が小さくなる
**原因**: jQueryのセレクタが親要素（`.main_area`）になっている
**対処**: 子要素（`.mainvisual_img`）に変更する
