# クラス名自動付与ルール

★HTMLタグに自動でクラス名を付与する命名規則

## 📐 命名規則

### 基本ルール
```
通常: [section]_[element]          (2単語)
状態: [section]_[element]_[state]  (3単語MAX)
```

### ★重要ルール
- ✅ 接続は `_` のみ使用
- ❌ 省略形禁止（`img` `nav` 以外）
- ❌ 番号は使わない（同じクラス使い回し）
- ✅ 既存クラス名は上書きOK（サジェスト等の仮クラス → 正しい命名で上書き）
- ✅ 画像は `<img>` → クラス名: `[section]_img`
- ✅ 明らかな文法ミスは自動修正（`/* */` → `<!-- -->`）
- ⚠️ 文法的におかしい構造 → ユーザー確認

---

## 例

### 入力（あなたが書く）
```html
<div>
  <h2>タイトル</h2>
  <p>説明文</p>
</div>
```

### 出力（AIが付与）
```html
<div class="accordion_area">
  <div class="accordion_item">
    <h2 class="accordion_title">タイトル</h2>
    <p class="accordion_description">説明文</p>
  </div>
</div>
```

---

## CSS例（状態管理）

```css
/* 閉じてる時 */
.accordion_description {
  display: none;
}

/* 開いてる時 */
.accordion_item_active .accordion_description {
  display: block;
}
```

---

## JavaScript例

```javascript
// クラス追加で状態変更
item.classList.toggle('accordion_item_active');
```

---

## ⚠️ 確認が必要なケース

### ケース1: ネスト深すぎ
```html
<div><div><div><div>4階層</div></div></div></div>
→ 命名困難。どうする？
```

### ケース2: 文法的におかしい
```html
<button><h1>見出し</h1></button>
→ ボタン内に見出しは変だけどOK？
```

### ケース3: 役割が不明
```html
<div><!-- 複雑な構造 --></div>
→ accordion_wrapper でいい？
```

---

## 📋 よく使う要素名

| 用途 | クラス名 |
|------|----------|
| 外枠 | `_area`（※`_container`は使わない） |
| 項目 | `_item` |
| 見出し | `_title` / `_header` |
| 本文 | `_description` / `_content` |
| ボタン | `_button` |
| アイコン | `_icon` |
| ラッパー | `_wrapper` |

---

## 🎯 運用フロー

1. あなた: `<タグ>` だけ書く
2. AI: `class="クラス名"` 自動付与
3. おかしい時: AIが確認
4. OK時: そのまま実装

## 📐 CSSセクションコメント（特徴付き）

```css
/* ┌─────────────────────────────────────────┐
   │ ヘッダー                                │
   │ - fixed上部固定                         │
   │ - flex横並び（ロゴ + ナビ + ボタン）     │
   └─────────────────────────────────────────┘ */

/* ┌─────────────────────────────────────────┐
   │ メインビジュアル                        │
   │ - スライドショー（JSで制御）            │
   │ - 全幅背景画像                          │
   │ - position: relative + absolute重ね     │
   └─────────────────────────────────────────┘ */

/* ┌─────────────────────────────────────────┐
   │ 診療案内                                │
   │ - flex 2カラム（皮膚科 / 美容皮膚科）   │
   │ - 背景画像 + テキスト重ね              │
   └─────────────────────────────────────────┘ */
```

★セクション名の先頭はHTMLと同じ名前を使用
★特徴・使用技術（flex, grid, position等）を記載
★不要なら人間が削除すればOK
