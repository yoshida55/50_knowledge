## ▢ スライドショーの仕組みまとめ（シンプル版）

「同じ位置に重ねた画像を、時間をずらして表示・非表示（ふわっと）切り替える」という仕組みです。

【HTML】
```html
<div class="bg_image">
  <div class="slide slide1"></div>
  <div class="slide slide2"></div>
  <div class="slide slide3"></div>
</div>
```

【CSS】
```css
/* 1. 外枠（すべての基準） */
.bg_image {
  position: relative;
  width: 100%;
  height: 72rem;
  overflow: hidden;
}

/* 2. 画像の上に「ぼかし（内側の影）」を重ねる */
.bg_image::after {
  content: "";
  position: absolute;
  top: 0; left: 0;
  width: 100%; height: 100%;
  box-shadow: inset 0 0 20px 20px #fff; /* 内側へのぼかし */
  z-index: 10;
  pointer-events: none; /* 下の要素をクリックできるようにする */
}

/* 3. 各スライドの共通設定 */
.slide {
  position: absolute; /* 同じ場所に重ねるために必須 */
  top: 0; left: 0;
  width: 100%; height: 100%;
  opacity: 0; /* 最初は消しておく */
  background-size: cover;
  background-position: center;
  animation: slideshow 15s infinite; /* 15秒で1ループを繰り返す */
}

/* 4. タイミングをずらす（重要） */
.slide1 { background-image: url("../img/mainvisual1.jpg"); animation-delay: 0s; }
.slide2 { background-image: url("../img/mainvisual2.jpg"); animation-delay: 5s; }
.slide3 { background-image: url("../img/mainvisual3.jpg"); animation-delay: 10s; }

/* 5. 「いつ出て、いつ消えるか」の指示 */
@keyframes slideshow {
  0%   { opacity: 0; }
  5%   { opacity: 1; }  /* 5%の地点で表示完了 */
  33%  { opacity: 1; }  /* 33%まで表示を維持 */
  38%  { opacity: 0; }  /* 38%で完全に消える */
  100% { opacity: 0; }
}
```
