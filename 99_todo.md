### コメントをしっかりかく
各セクションのコメントは以下のコメントを以下形式でかく


【例】HTML
```html
<!-- ===============================================
 Gallery （flex）画像が左右、階段式にならぶ
 =============================================== -->
```

【例】CSS
```css
/* ┌─────────────────────────────────────────┐
   │ Gallery （flex）画像が左右、階段式にならぶ                        │
   └─────────────────────────────────────────┘ */
```

`あとで見て直感的にわかるように、かく詳しくかく`
